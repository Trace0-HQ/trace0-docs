---
title: What is Trace0?
description: Understand what Trace0 does and how it helps you debug and monitor your systems.
---

Trace0 is an modern, easy-to-use observability platform that gives you traces, metrics, logs, and alerting in one place — without the complexity or cost of traditional monitoring tools.

Trace0 provides the following features:

<CardGroup cols={2}>
  <Card title="Tracing" icon="route" href="/platform/transactions">
    Distributed tracing out of the box. Get full visibility into requests across multiple services, and see detailed activity within each service — including database queries, external calls, and request timings.
  </Card>
  <Card title="Logging" icon="file-lines" href="/platform/logs">
    Powerful log management with full context. Search logs by free text or time range, and instantly view the complete request trace for any log entry.
  </Card>
  <Card title="Metrics" icon="chart-line" href="/platform/metrics">
    Detailed metrics for each service. Monitor request latency, error rates, memory usage, and more — all displayed in clear, easy-to-read dashboards.
  </Card>
  <Card title="Alerting" icon="bell" href="/platform/alerts">
    Create alerts for errors and metrics. Trigger alerts when a service fails or exceeds memory thresholds, and receive instant notifications via email or Slack.
  </Card>
  <Card title="OpenTelemetry-native" icon="code" href="/faq/what-is-opentelemetry">
    Trace0 is built on OpenTelemetry, so you can instrument your services using standard, open-source libraries with minimal setup.
  </Card>
  <Card title="Simple Pricing" icon="tag" href="/pricing/pricing">
    Simple, usage-based pricing. You pay **$0.60 per GB** of ingested data — nothing else. 30-day retention, all features included. No seat fees, no per-service charges, no surprises.
  </Card>
</CardGroup>
