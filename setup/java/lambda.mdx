---
title: Java AWS Lambda
description: Instrument a Java Lambda function with OpenTelemetry and send traces to Trace0.
---

<Note>
  You only need to add the **JavaWrapper layer** — no separate collector layer is required. Trace0 handles ingestion directly.
</Note>

## 1. Add the OpenTelemetry Lambda Layer

Find the latest `layer-javaagent` ARN for your AWS region from the [opentelemetry-lambda releases page](https://github.com/open-telemetry/opentelemetry-lambda/releases). Add the **JavaWrapper** layer ARN to your function — do **not** add the Collector layer.

## 2. Set Environment Variables

Add the following environment variables to your Lambda function. Replace `YOUR_TRACEAPP_ENV_API_KEY` with the API key from your Trace0 environment.

| Variable | Value |
|---|---|
| `AWS_LAMBDA_EXEC_WRAPPER` | See [handler table below](#choose-the-right-exec-wrapper) |
| `OTEL_EXPORTER_OTLP_PROTOCOL` | `http/protobuf` |
| `OTEL_EXPORTER_OTLP_ENDPOINT` | `https://app.trace0hq.com/api` |
| `OTEL_EXPORTER_OTLP_HEADERS` | `X-API-KEY=YOUR_TRACEAPP_ENV_API_KEY` |

### Choose the Right Exec Wrapper

The value of `AWS_LAMBDA_EXEC_WRAPPER` depends on how your function is triggered. See the [opentelemetry-lambda Java README](https://github.com/open-telemetry/opentelemetry-lambda/blob/main/java/README.md) for full details.

| Trigger | `AWS_LAMBDA_EXEC_WRAPPER` value |
|---|---|
| API Gateway (proxy integration) | `/opt/otel-proxy-handler` |
| SQS | `/opt/otel-sqs-handler` |
| Streaming (`RequestStreamHandler`) | `/opt/otel-stream-handler` |
| All other triggers (EventBridge, S3, SNS, etc.) | `/opt/otel-handler` — use the **JavaAgent** layer ARN |

## 3. Configure Your Infrastructure

<Tabs>
  <Tab title="CloudFormation">
    ```yaml
    Parameters:
      OtelJavaLayerArn:
        Type: String
        Description: >
          JavaWrapper layer ARN from https://github.com/open-telemetry/opentelemetry-lambda/releases
          e.g. arn:aws:lambda:us-east-1:184161586896:layer:opentelemetry-javaagent-0_11_0:1
      Trace0ApiKey:
        Type: String
        NoEcho: true

    Resources:
      MyFunction:
        Type: AWS::Lambda::Function
        Properties:
          FunctionName: my-java-function
          Runtime: java21
          Handler: com.example.Handler::handleRequest
          Code:
            S3Bucket: !Ref DeploymentBucket
            S3Key: function.jar
          Layers:
            - !Ref OtelJavaLayerArn
          Environment:
            Variables:
              AWS_LAMBDA_EXEC_WRAPPER: /opt/otel-proxy-handler
              OTEL_EXPORTER_OTLP_PROTOCOL: http/protobuf
              OTEL_EXPORTER_OTLP_ENDPOINT: https://app.trace0hq.com/api
              OTEL_EXPORTER_OTLP_HEADERS: !Sub "X-API-KEY=${Trace0ApiKey}"
    ```
  </Tab>
  <Tab title="Terraform">
    ```hcl
    variable "otel_java_layer_arn" {
      type        = string
      description = "JavaWrapper layer ARN from https://github.com/open-telemetry/opentelemetry-lambda/releases"
    }

    variable "trace0_api_key" {
      type      = string
      sensitive = true
    }

    resource "aws_lambda_function" "app" {
      function_name = "my-java-function"
      runtime       = "java21"
      handler       = "com.example.Handler::handleRequest"
      filename      = "function.jar"

      layers = [var.otel_java_layer_arn]

      environment {
        variables = {
          AWS_LAMBDA_EXEC_WRAPPER       = "/opt/otel-proxy-handler"
          OTEL_EXPORTER_OTLP_PROTOCOL  = "http/protobuf"
          OTEL_EXPORTER_OTLP_ENDPOINT  = "https://app.trace0hq.com/api"
          OTEL_EXPORTER_OTLP_HEADERS   = "X-API-KEY=${var.trace0_api_key}"
        }
      }
    }
    ```
  </Tab>
  <Tab title="CDK (TypeScript)">
    ```typescript
    import * as cdk from 'aws-cdk-lib';
    import * as lambda from 'aws-cdk-lib/aws-lambda';
    import * as secretsmanager from 'aws-cdk-lib/aws-secretsmanager';

    // Find the latest layer ARN at:
    // https://github.com/open-telemetry/opentelemetry-lambda/releases
    const otelLayer = lambda.LayerVersion.fromLayerVersionArn(
      this, 'OtelJavaLayer',
      'arn:aws:lambda:us-east-1:184161586896:layer:opentelemetry-javaagent-0_11_0:1'
    );

    const apiKeySecret = secretsmanager.Secret.fromSecretNameV2(
      this, 'Trace0ApiKey', 'trace0/api-key'
    );

    new lambda.Function(this, 'MyFunction', {
      runtime: lambda.Runtime.JAVA_21,
      handler: 'com.example.Handler::handleRequest',
      code: lambda.Code.fromAsset('function.jar'),
      layers: [otelLayer],
      environment: {
        AWS_LAMBDA_EXEC_WRAPPER:      '/opt/otel-proxy-handler',
        OTEL_EXPORTER_OTLP_PROTOCOL:  'http/protobuf',
        OTEL_EXPORTER_OTLP_ENDPOINT:  'https://app.trace0hq.com/api',
        OTEL_EXPORTER_OTLP_HEADERS:   `X-API-KEY=${apiKeySecret.secretValue.unsafeUnwrap()}`,
      },
    });
    ```

  <Tab title="GCP Cloud Functions">
    <Note>
      GCP Cloud Functions does not use Lambda layers. Instead, include the OpenTelemetry Java agent JAR in your deployment and configure it via environment variables.
    </Note>

    Download the [opentelemetry-javaagent.jar](https://github.com/open-telemetry/opentelemetry-java-instrumentation/releases) and include it in your function package, then set:

    ```hcl
    # Terraform — GCP Cloud Functions (2nd gen)
    variable "trace0_api_key" {
      type      = string
      sensitive = true
    }

    resource "google_cloudfunctions2_function" "app" {
      name     = "my-java-function"
      location = "us-central1"

      build_config {
        runtime     = "java21"
        entry_point = "com.example.Handler"
        source {
          storage_source {
            bucket = google_storage_bucket.source.name
            object = google_storage_bucket_object.source.name
          }
        }
      }

      service_config {
        environment_variables = {
          JAVA_TOOL_OPTIONS            = "-javaagent:opentelemetry-javaagent.jar"
          OTEL_EXPORTER_OTLP_PROTOCOL = "http/protobuf"
          OTEL_EXPORTER_OTLP_ENDPOINT = "https://app.trace0hq.com/api"
        }
        secret_environment_variables {
          key        = "OTEL_EXPORTER_OTLP_HEADERS"
          project_id = var.project_id
          secret     = google_secret_manager_secret.trace0_api_key.secret_id
          version    = "latest"
        }
      }
    }
    ```

    Set the secret value to `X-API-KEY=YOUR_TRACEAPP_ENV_API_KEY`.
