---
title: Go — AWS Lambda
description: Instrument a Go Lambda function with OpenTelemetry and send traces to Trace0.
---

<Note>
  You only need to add the **Go Lambda layer** — no separate collector layer is required. Trace0 handles ingestion directly.
</Note>

## 1. Add the OpenTelemetry Lambda Layer

Find the latest `layer-collector` ARN for Go in your AWS region from the [opentelemetry-lambda releases page](https://github.com/open-telemetry/opentelemetry-lambda/releases). Add the layer ARN to your function.

## 2. Instrument Your Handler

Install the Go Lambda wrapper:

```bash
go get go.opentelemetry.io/contrib/instrumentation/github.com/aws/aws-lambda-go/otellambda
```

Wrap your handler with `otellambda.InstrumentHandler`:

```go main.go
package main

import (
    "github.com/aws/aws-lambda-go/lambda"
    "go.opentelemetry.io/contrib/instrumentation/github.com/aws/aws-lambda-go/otellambda"
)

func handler(ctx context.Context, event MyEvent) (string, error) {
    // your handler logic
    return "ok", nil
}

func main() {
    lambda.Start(otellambda.InstrumentHandler(handler))
}
```

## 3. Set Environment Variables

| Variable | Value |
|---|---|
| `OTEL_EXPORTER_OTLP_PROTOCOL` | `http/protobuf` |
| `OTEL_EXPORTER_OTLP_ENDPOINT` | `https://app.trace0hq.com/api` |
| `OTEL_EXPORTER_OTLP_HEADERS` | `X-API-KEY=YOUR_TRACEAPP_ENV_API_KEY` |

## 4. Deploy Service
Deploy your service and watch traces, metrics, and logs appear in the [Trace0 dashboard](https://app.trace0hq.com) within seconds.

## Sample Code
See Infrastructure as Code (IaC) examples below for adding the OpenTelemetry Go Lambda Layer and Trace0 environment variables to your service:

<Tabs>
  <Tab title="CloudFormation">
    ```yaml
    Parameters:
      OtelGoLayerArn:
        Type: String
        Description: >
          Go layer ARN from https://github.com/open-telemetry/opentelemetry-lambda/releases
      Trace0ApiKey:
        Type: String
        NoEcho: true

    Resources:
      MyFunction:
        Type: AWS::Lambda::Function
        Properties:
          FunctionName: my-go-function
          Runtime: provided.al2023
          Handler: bootstrap
          Code:
            S3Bucket: !Ref DeploymentBucket
            S3Key: function.zip
          Layers:
            - !Ref OtelGoLayerArn
          Environment:
            Variables:
              OTEL_EXPORTER_OTLP_PROTOCOL: http/protobuf
              OTEL_EXPORTER_OTLP_ENDPOINT: https://app.trace0hq.com/api
              OTEL_EXPORTER_OTLP_HEADERS:  !Sub "X-API-KEY=${Trace0ApiKey}"
    ```
  </Tab>
  <Tab title="Terraform">
    ```hcl
    variable "otel_go_layer_arn" {
      type        = string
      description = "Go layer ARN from https://github.com/open-telemetry/opentelemetry-lambda/releases"
    }

    variable "trace0_api_key" {
      type      = string
      sensitive = true
    }

    resource "aws_lambda_function" "app" {
      function_name = "my-go-function"
      runtime       = "provided.al2023"
      handler       = "bootstrap"
      filename      = "function.zip"

      layers = [var.otel_go_layer_arn]

      environment {
        variables = {
          OTEL_EXPORTER_OTLP_PROTOCOL = "http/protobuf"
          OTEL_EXPORTER_OTLP_ENDPOINT = "https://app.trace0hq.com/api"
          OTEL_EXPORTER_OTLP_HEADERS  = "X-API-KEY=${var.trace0_api_key}"
        }
      }
    }
    ```
  </Tab>
  <Tab title="CDK (TypeScript)">
    ```typescript
    import * as lambda from 'aws-cdk-lib/aws-lambda';

    // Find the latest layer ARN at:
    // https://github.com/open-telemetry/opentelemetry-lambda/releases
    const otelLayer = lambda.LayerVersion.fromLayerVersionArn(
      this, 'OtelGoLayer',
      'arn:aws:lambda:us-east-1:184161586896:layer:opentelemetry-collector-0_11_0:1'
    );

    new lambda.Function(this, 'MyFunction', {
      runtime: lambda.Runtime.PROVIDED_AL2023,
      handler: 'bootstrap',
      code: lambda.Code.fromAsset('function.zip'),
      layers: [otelLayer],
      environment: {
        OTEL_EXPORTER_OTLP_PROTOCOL: 'http/protobuf',
        OTEL_EXPORTER_OTLP_ENDPOINT: 'https://app.trace0hq.com/api',
        OTEL_EXPORTER_OTLP_HEADERS:  `X-API-KEY=${apiKey}`,
      },
    });
    ```
  </Tab>
  <Tab title="GCP Cloud Functions">
    <Note>
      GCP Cloud Functions does not use Lambda layers. The OTel SDK is compiled directly into your binary — no extra setup is needed beyond the environment variables.
    </Note>

    ```hcl
    resource "google_cloudfunctions2_function" "app" {
      name     = "my-go-function"
      location = "us-central1"

      build_config {
        runtime     = "go123"
        entry_point = "Handler"
        source {
          storage_source {
            bucket = google_storage_bucket.source.name
            object = google_storage_bucket_object.source.name
          }
        }
      }

      service_config {
        environment_variables = {
          OTEL_EXPORTER_OTLP_ENDPOINT = "https://app.trace0hq.com/api"
          OTEL_EXPORTER_OTLP_PROTOCOL = "http/protobuf"
        }
        secret_environment_variables {
          key        = "OTEL_EXPORTER_OTLP_HEADERS"
          project_id = var.project_id
          secret     = google_secret_manager_secret.trace0_api_key.secret_id
          version    = "latest"
        }
      }
    }
    ```

    Set the secret value to `X-API-KEY=YOUR_TRACEAPP_ENV_API_KEY`.
