---
title: Python — AWS Lambda
description: Instrument a Python Lambda function with OpenTelemetry and send traces to Trace0.
---

<Note>
  You only need to add the **Python Lambda layer** — no separate collector layer is required. Trace0 handles ingestion directly.
</Note>

## 1. Add the OpenTelemetry Lambda Layer

Find the latest `layer-python` ARN for your AWS region from the [opentelemetry-lambda releases page](https://github.com/open-telemetry/opentelemetry-lambda/releases). Add the layer ARN to your function.

## 2. Set Environment Variables

| Variable | Value |
|---|---|
| `AWS_LAMBDA_EXEC_WRAPPER` | `/opt/otel-instrument` |
| `OTEL_EXPORTER_OTLP_PROTOCOL` | `http/protobuf` |
| `OTEL_EXPORTER_OTLP_ENDPOINT` | `https://app.trace0hq.com/api` |
| `OTEL_EXPORTER_OTLP_HEADERS` | `X-API-KEY=YOUR_TRACEAPP_ENV_API_KEY` |

## 3. Configure Your Infrastructure

<Tabs>
  <Tab title="CloudFormation">
    ```yaml
    Parameters:
      OtelPythonLayerArn:
        Type: String
        Description: >
          Python layer ARN from https://github.com/open-telemetry/opentelemetry-lambda/releases
      Trace0ApiKey:
        Type: String
        NoEcho: true

    Resources:
      MyFunction:
        Type: AWS::Lambda::Function
        Properties:
          FunctionName: my-python-function
          Runtime: python3.13
          Handler: handler.lambda_handler
          Code:
            S3Bucket: !Ref DeploymentBucket
            S3Key: function.zip
          Layers:
            - !Ref OtelPythonLayerArn
          Environment:
            Variables:
              AWS_LAMBDA_EXEC_WRAPPER:      /opt/otel-instrument
              OTEL_EXPORTER_OTLP_PROTOCOL:  http/protobuf
              OTEL_EXPORTER_OTLP_ENDPOINT:  https://app.trace0hq.com/api
              OTEL_EXPORTER_OTLP_HEADERS:   !Sub "X-API-KEY=${Trace0ApiKey}"
    ```
  </Tab>
  <Tab title="Terraform">
    ```hcl
    variable "otel_python_layer_arn" {
      type        = string
      description = "Python layer ARN from https://github.com/open-telemetry/opentelemetry-lambda/releases"
    }

    variable "trace0_api_key" {
      type      = string
      sensitive = true
    }

    resource "aws_lambda_function" "app" {
      function_name = "my-python-function"
      runtime       = "python3.13"
      handler       = "handler.lambda_handler"
      filename      = "function.zip"

      layers = [var.otel_python_layer_arn]

      environment {
        variables = {
          AWS_LAMBDA_EXEC_WRAPPER      = "/opt/otel-instrument"
          OTEL_EXPORTER_OTLP_PROTOCOL = "http/protobuf"
          OTEL_EXPORTER_OTLP_ENDPOINT = "https://app.trace0hq.com/api"
          OTEL_EXPORTER_OTLP_HEADERS  = "X-API-KEY=${var.trace0_api_key}"
        }
      }
    }
    ```
  </Tab>
  <Tab title="CDK (TypeScript)">
    ```typescript
    import * as lambda from 'aws-cdk-lib/aws-lambda';

    // Find the latest layer ARN at:
    // https://github.com/open-telemetry/opentelemetry-lambda/releases
    const otelLayer = lambda.LayerVersion.fromLayerVersionArn(
      this, 'OtelPythonLayer',
      'arn:aws:lambda:us-east-1:184161586896:layer:opentelemetry-python-0_11_0:1'
    );

    new lambda.Function(this, 'MyFunction', {
      runtime: lambda.Runtime.PYTHON_3_13,
      handler: 'handler.lambda_handler',
      code: lambda.Code.fromAsset('function.zip'),
      layers: [otelLayer],
      environment: {
        AWS_LAMBDA_EXEC_WRAPPER:      '/opt/otel-instrument',
        OTEL_EXPORTER_OTLP_PROTOCOL:  'http/protobuf',
        OTEL_EXPORTER_OTLP_ENDPOINT:  'https://app.trace0hq.com/api',
        OTEL_EXPORTER_OTLP_HEADERS:   `X-API-KEY=${apiKey}`,
      },
    });
    ```
  </Tab>
  <Tab title="GCP Cloud Functions">
    <Note>
      GCP Cloud Functions does not use Lambda layers. Add the OpenTelemetry packages to your `requirements.txt` and initialise the SDK at import time.
    </Note>

    ```bash requirements.txt
    opentelemetry-distro
    opentelemetry-exporter-otlp
    opentelemetry-instrumentation-flask
    ```

    ```hcl
    resource "google_cloudfunctions2_function" "app" {
      name     = "my-python-function"
      location = "us-central1"

      build_config {
        runtime     = "python313"
        entry_point = "handler"
        source {
          storage_source {
            bucket = google_storage_bucket.source.name
            object = google_storage_bucket_object.source.name
          }
        }
      }

      service_config {
        environment_variables = {
          OTEL_EXPORTER_OTLP_ENDPOINT = "https://app.trace0hq.com/api"
          OTEL_EXPORTER_OTLP_PROTOCOL = "http/protobuf"
        }
        secret_environment_variables {
          key        = "OTEL_EXPORTER_OTLP_HEADERS"
          project_id = var.project_id
          secret     = google_secret_manager_secret.trace0_api_key.secret_id
          version    = "latest"
        }
      }
    }
    ```

    Set the secret value to `X-API-KEY=YOUR_TRACEAPP_ENV_API_KEY`.
